## Что такое традиционное монгольское письмо?

Традиционное монгольское письмо (ᠮᠣᠩᠭᠣᠯ ᠪᠢᠴᠢᠭ᠌, Монгол бичиг) — это классическая и традиционная система письма монгольских языков. Оно имеет более чем 800-летнюю историю и является одним из немногих вертикальных письменностей, которые используются до сих пор. Письмо пишется вертикально сверху вниз, а столбцы идут слева направо.

Сегодня традиционное монгольское письмо:
- Является официальным письмом Внутренней Монголии (КНР)
- Переживает возрождение в Монголии, где оно вводится наряду с кириллицей
- Используется монгольскими общинами по всему миру для сохранения культурного наследия
- Поддерживается в Unicode (U+1800–U+18AF) и современных цифровых системах

## Конвертеры

Инструменты для конвертации между различными письменностями и кодировками.

- [mongol_converter](https://github.com/suragch/mongol_converter) - Конвертер из кириллицы в монгольское письмо, созданный на Flutter. Backend размещен на PocketBase.

## Шрифты

Шрифты с открытым исходным кодом, поддерживающие традиционное монгольское письмо.

- [Noto Mongolian](https://github.com/notofonts/mongolian) - Шрифт для монгольского письма из семейства Noto от Google, часть проекта Noto Fonts.
