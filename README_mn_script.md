> ᠤᠯᠠᠮᠵᠢᠯᠠᠯᠲᠤ ᠮᠣᠩᠭᠣᠯ ᠪᠢᠴᠢᠭ᠌ ᠦᠨ (ᠮᠣᠩᠭᠣᠯ ᠪᠢᠴᠢᠭ᠌) ᠨᠡᠭᠡᠭᠡᠯᠲᠡᠲᠡᠢ ᠡᠬᠢ ᠶᠢᠨ ᠲᠥᠰᠦᠯ᠂ ᠰᠠᠩ᠂ ᠬᠡᠷᠡᠭᠰᠦᠯ ᠪᠣᠯᠣᠨ ᠨᠥᠭᠥᠴᠡ ᠶᠢᠨ ᠰᠣᠩᠭᠣᠮᠠᠯ ᠵᠠᠭᠰᠠᠭᠠᠯᠲᠠ᠃

## ᠤᠯᠠᠮᠵᠢᠯᠠᠯᠲᠤ ᠮᠣᠩᠭᠣᠯ ᠪᠢᠴᠢᠭ᠌ ᠭᠡᠵᠦ ᠶᠤᠤ ᠪᠤᠢ?

ᠤᠯᠠᠮᠵᠢᠯᠠᠯᠲᠤ ᠮᠣᠩᠭᠣᠯ ᠪᠢᠴᠢᠭ᠌ (ᠮᠣᠩᠭᠣᠯ ᠪᠢᠴᠢᠭ᠌᠂ ᠮᠣᠩᠭᠣᠯ ᠪᠢᠴᠢᠭ᠌) ᠨᠢ ᠮᠣᠩᠭᠣᠯ ᠬᠡᠯᠡᠨ ᠦ ᠰᠣᠩᠭᠣᠳᠣᠭ ᠪᠥᠭᠡᠳ ᠤᠯᠠᠮᠵᠢᠯᠠᠯᠲᠤ ᠪᠢᠴᠢᠭ᠌ ᠦᠨ ᠰᠢᠰᠲ᠋ᠧᠮ ᠶᠤᠮ᠃ ᠡᠨᠡ ᠨᠢ 800 ᠭᠠᠷᠤᠢ ᠵᠢᠯ ᠦᠨ ᠲᠡᠦᠬᠡᠲᠡᠢ ᠪᠥᠭᠡᠳ ᠥᠨᠥᠭᠡ ᠬᠦᠷᠲᠡᠯ᠎ᠡ ᠠᠰᠢᠭᠯᠠᠭᠳᠠᠵᠤ ᠪᠠᠶᠢᠭ᠎ᠠ ᠴᠥᠭᠡᠨ ᠬᠡᠳᠦᠨ ᠪᠣᠰᠤᠭᠤ ᠪᠢᠴᠢᠭ᠌ ᠦᠨ ᠨᠢᠭᠡ ᠶᠤᠮ᠃ ᠪᠢᠴᠢᠭ᠌ ᠨᠢ ᠳᠡᠭᠡᠷ᠎ᠡ ᠡᠴᠡ ᠳᠣᠣᠷᠠᠭᠰᠢ ᠪᠣᠰᠤᠭᠤ ᠪᠢᠴᠢᠭᠳᠡᠵᠦ᠂ ᠪᠠᠭᠠᠨ᠎ᠠ ᠨᠢ ᠵᠡᠭᠦᠨ ᠡᠴᠡ ᠪᠠᠷᠠᠭᠤᠨ ᠲᠢᠶᠡᠰᠢ ᠴᠢᠭᠯᠡᠨ᠎ᠡ᠃

ᠥᠨᠥᠳᠥᠷ ᠤᠯᠠᠮᠵᠢᠯᠠᠯᠲᠤ ᠮᠣᠩᠭᠣᠯ ᠪᠢᠴᠢᠭ᠌:

- ᠥᠪᠥᠷ ᠮᠣᠩᠭᠣᠯ ᠤᠨ (ᠪᠦᠭᠦᠳᠡ ᠨᠠᠶᠢᠷᠠᠮᠳᠠᠬᠤ ᠳᠤᠮᠳᠠᠳᠤ ᠠᠷᠠᠳ ᠤᠯᠤᠰ) ᠠᠯᠪᠠᠨ ᠶᠣᠰᠤᠨ ᠤ ᠪᠢᠴᠢᠭ᠌ ᠶᠤᠮ
- ᠮᠣᠩᠭᠣᠯ ᠤᠯᠤᠰ ᠲᠤ ᠰᠡᠷᠭᠡᠨ ᠮᠠᠨᠳᠤᠵᠤ᠂ ᠺᠢᠷᠢᠯ ᠦᠰᠦᠭ ᠦᠨ ᠬᠠᠵᠠᠭᠤᠳᠠ ᠨᠡᠪᠲᠡᠷᠦᠯᠡᠭᠳᠡᠵᠦ ᠪᠠᠶᠢᠨ᠎ᠠ
- ᠳᠡᠯᠡᠬᠡᠢ ᠳᠠᠶᠠᠭᠠᠷ ᠮᠣᠩᠭᠣᠯ ᠬᠦᠮᠦᠰ ᠰᠣᠶᠤᠯ ᠤᠨ ᠥᠪ ᠢ ᠬᠠᠳᠠᠭᠠᠯᠠᠬᠤ ᠶᠢᠨ ᠲᠤᠯᠠᠳᠠ ᠠᠰᠢᠭᠯᠠᠳᠠᠭ
- Unicode (U+1800–U+18AF) ᠪᠣᠯᠣᠨ ᠣᠷᠴᠢᠨ ᠦᠶ᠎ᠡ ᠶᠢᠨ ᠳ᠋ᠢᠵᠢᠲ᠋ᠠᠯ ᠰᠢᠰᠲ᠋ᠧᠮ ᠳᠦ ᠳᠡᠮᠵᠢᠭᠳᠡᠭᠳᠡᠭ
